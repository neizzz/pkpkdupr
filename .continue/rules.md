## 규칙

모든 요청에 응답한 후, 답변의 맨 아래에 **소요시간**을 적는다.

- 형식: `⏱️ 소요시간: {분}분 {초}초` 또는 `⏱️ 소요시간: {초}초(<1분인 경우)`
- 작업이 복잡하거나 여러 단계를 거친 경우 각 단계별 소요시간과 총 시간을 함께 기재한다.

**중요 — 소요시간의 정확성:**
소요시간은 **실제 대화 시작 시점부터 응답 완료 시점까지의 경과 시간**을 기준으로 기록한다. 요약(job)에서 계산된 시간이 아니라, 툴 호출 횟수, 파일 읽기/쓰기 횟수, 브랜치 확인 등 실제 작업 내용을 종합해서 판단하고 적는다.

- 단순 질문 답변만: 5~30초
- 파일 1~2개 읽고 답변: 30~60초
- 스킬/설정 파일 생성 또는 여러 파일 편집: 1~5분
- 여러 툴 호출 + 분석 + 생성 (예: 프로젝트 탐색 → 분석 → 파일 여러 개 작성): 3~10분
